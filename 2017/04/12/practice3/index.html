  1. 实验三 使用Github管理Android项目

    一、实验目的

    1. 掌握Android Studio中Git使用

    2. 掌握Github管理项目

    二、实验任务

    1. 在Android Studio中新建项目

    2. 添加VCS中的Git支持

    3. 发布项目到Github

    4. 在Android Studio中进行修改代码、提交、新建分支、合并等操作

    5. 推送项目到Github

    6. 如有多人合作，处理pull request

    三、实验步骤与结果

    1. 在Android Studio中建立你的项目

    2. 打开你的Github

    3. 添加VCS中的Git支持

    1. 发布项目到Github

    1. 在Android Studio中进行修改代码，提交，新建分支，合并等操作

    修改代码和提交：

    可以勾选上右侧的Reformatcode（重新格式化），Optimizeimports（完善import），不勾选Perform code analysis

    1. Push到Github的结果：

    1. 如果有多人合作，就push request

    四、实验小结

    ​ 使用Github来管理安卓项目的好处有，无论你走到哪里，都可以查看你的项目，并且你的每一次修改，提交都被记录下来，有利于你的复看。还有就是，如果你的项目是多人合作的，只需要push request，就能复制到你的计算机，然后进行下一步的开发。
